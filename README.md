# Telecom X - Análisis de Churn

##  Descripción
Análisis exploratorio de datos para identificar factores asociados a la evasión de clientes.

##  Tecnologías
- Python
- Pandas
- Matplotlib
- Seaborn

##  Cómo ejecutar
1. Clonar repositorio
2. Instalar dependencias
3. Ejecutar notebook

##  Resultados
Se identificó que el tiempo de contrato y el tipo de contrato son los principales factores asociados al churn.
